## How To Make Screenshots Look Better

Keep the images truthful. Chrome wants screenshots to reflect the actual experience, so polish them lightly rather than turning them into posters.

### Do

- Use a demo account, not your personal email
- Replace real credentials with fake sample data
- Crop out the Windows taskbar and unrelated browser tabs
- Keep the Vestiga UI centered and readable
- Use consistent browser zoom so text looks sharp
- Use the same browser theme across all screenshots
- Keep the teal branding consistent
- Redact personal email addresses if needed

### Avoid

- Fake UI elements that don't exist in the product
- Huge marketing banners covering the product
- Busy browser bookmarks bars
- Personal accounts, real passwords, or personal data
- Tiny unreadable text
- Too many screenshots of the same screen

## Recommended Editing Style

If you edit the screenshots, keep it minimal:

- Add a soft top-left title strip only
- Use Vestiga teal as the accent color
- Use white or very light neutral background for the title strip
- Keep the title to 2 to 5 words
- Don't place text over critical UI
- Don't add phone or laptop device frames

## Search Ranking Reality

You should optimize strongly, but don't keyword-stuff. Chrome ranking is influenced by:

- title and description relevance
- listing quality
- screenshots and promo images
- install conversion rate
- uninstall rate
- ratings and reviews
- user engagement and retention

You can improve your odds for searches like `vault`, `security`, and `password manager`, but nobody can honestly promise top rank for those broad keywords at launch.
